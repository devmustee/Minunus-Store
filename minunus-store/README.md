# Minunus Store

**Minunus Store** is a streamlined web application designed for wood inventory management, sales tracking, and operational monitoring for small-scale wood stores.

## 🎯 Goal Statement
Minunus Store is designed to manage wood inventory, track sales and purchases, and monitor funds for a small operation. It provides two user roles:
*   **Admin**: Can add stock, set purchase prices, and view all sales and inventory across the entire system.
*   **Client**: Can record sales and view their own transactions securely.

The system ensures accurate tracking of product quantities, revenue, and remaining stock, simplifies reporting, and enables efficient, transparent management of the store’s operations.

## ✨ Key Features
- **Modern Premium Interface**: A high-performance, dark-themed dashboard with glassmorphism aesthetics.
- **Role-Based Access Control**: Secure separation between administrative management and client sales tools.
- **Inventory Ledger**: Real-time tracking of unit counts, sizing, and pricing for all wood assets.
- **Sales Monitoring**: Instant recording of transactions with automatic settlement calculation.
- **Responsive Navigation**: Optimized for both mobile devices (for floor sales) and desktop (for management).

## 🚀 Getting Started
1. **Login**: Use the "Quick Demo Access" buttons on the landing page.
2. **Catalog**: View live stock in the "Inventory Ledger".
3. **Sales**: Record a new transaction in the "Quick Transaction" portal.
4. **Admin**: (Admin only) Manage refills and prices in the "Management" view.
